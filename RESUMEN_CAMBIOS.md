```bash
# 4. Abre en navegador

```

## 📝 Próximos Pasos para Personalizar

1. **Información de Contacto**
   - Archivo: `components/ContactForm.jsx`
   - Reemplaza número WhatsApp (línea ~30)
   - Actualiza email y teléfono

2. **Imágenes**
   - Reemplaza URLs en `Gallery.jsx` con tus imágenes
   - O descarga de Unsplash, Pexels, Pixabay

3. **Colores**
   - Archivo: `styles/App.css`
   - Sección `:root` (líneas 1-10)

4. **Servicios**
   - Archivo: `components/Services.jsx`
   - Array `services` (línea ~3)

5. **Testimonios**
   - Archivo: `components/Team.jsx`
   - Array `testimonials` (línea ~3)
